# Umbrales de aprobacion

Esta referencia resume los umbrales usados en la sesion para compras internas.

| Importe | Circuito |
| --- | --- |
| Menos de 500 EUR | Manager directo |
| Entre 500 y 2.000 EUR | Responsable de area y validacion de Finanzas |
| Mas de 2.000 EUR | Direccion y orden de compra formal |

Toda compra de hardware debe incluir justificacion de necesidad, centro de coste y proveedor recomendado.
