# Criterios de priorizacion regional

Una region puede considerarse prioritaria cuando combina varias senales:

- Pipeline relevante que podria desbloquearse con la solicitud.
- Margen suficiente para absorber el coste operativo.
- Conversion razonable frente a otras regiones.
- Urgencia operativa claramente descrita por el usuario.

Estos criterios no reemplazan las metricas reales. Sirven para explicar como leerlas y para evitar recomendaciones basadas solo en intuicion.
