# 1. Principios

Todo gasto corporativo debe ser necesario, proporcional, documentado y aprobado por la persona responsable. La compañía puede rechazar gastos sin justificante válido, fuera de política o no relacionados con la actividad profesional.

# 2. Gastos reembolsables

Se consideran reembolsables los gastos de viaje, dietas, transporte, alojamiento, material autorizado y otros conceptos aprobados previamente. El empleado debe presentar factura o justificante con fecha, proveedor, importe, impuestos y concepto.

Los tickets simplificados solo se aceptan cuando la normativa lo permite y el importe no supera el límite publicado en el portal.

# 3. Proceso de liquidación de gastos

1. El empleado registra el gasto en **Portal de empleados > Finanzas > Mis gastos**.
2. Adjunta factura o justificante.
3. Indica centro de coste, proyecto y motivo.
4. El manager aprueba necesidad y proporcionalidad.
5. Finanzas valida requisitos fiscales.
6. El reembolso se incluye en la siguiente liquidación disponible.

Los gastos deben presentarse antes de treinta días naturales desde la fecha de emisión. Gastos presentados fuera de plazo requieren justificación adicional.

# 4. Compras y proveedores

## 4.1 Solicitud de compra

Toda compra superior al umbral definido debe iniciarse con una solicitud de compra. La solicitud debe incluir necesidad, presupuesto estimado, proveedor propuesto, alternativas evaluadas y fecha requerida.

No se permite fraccionar compras para evitar aprobaciones. Si la compra afecta a software, datos, seguridad o sistemas corporativos, IT y Seguridad deben revisar antes de contratar.

## 4.2 Alta de proveedor

El alta de proveedor requiere datos fiscales, contacto administrativo, certificación bancaria, condiciones de pago, política de privacidad y aceptación del código de conducta de proveedores.

Finanzas no procesará facturas de proveedores no dados de alta o con datos incompletos.

# 5. Matriz de aprobación

| Importe estimado | Aprobación requerida |
|---|---|
| Hasta 500 EUR | Manager directo |
| 501 a 3.000 EUR | Manager y responsable de área |
| 3.001 a 15.000 EUR | Dirección de área y Finanzas |
| Más de 15.000 EUR | Comité de inversión |

Los importes son orientativos y pueden variar por unidad de negocio. El portal aplica automáticamente la matriz vigente.

# 6. Tarjetas corporativas

Las tarjetas corporativas se asignan solo a perfiles con necesidad recurrente. No se usan para gastos personales, retiradas de efectivo ni compras no autorizadas. Cada movimiento debe justificarse en el portal.

# 7. Facturas

Las facturas deben emitirse a nombre de NovaCorp Iberia S.L. e incluir número fiscal, fecha, base imponible, impuestos y descripción del servicio. Las facturas con errores se devuelven al proveedor para corrección.

# 8. Excepciones

Las excepciones se documentan antes de incurrir en el gasto. Si la excepción ocurre durante un viaje o emergencia operativa, se justifica al registrar el gasto y requiere aprobación de Finanzas.

# 9. Criterios de escalado

Se escala a Finanzas si existe duda fiscal, proveedor bloqueado, factura rechazada o gasto fuera de política. Se escala a Compras si se requiere negociación con proveedor. Se escala a IT o Seguridad si la compra afecta a tecnología, datos o accesos.
